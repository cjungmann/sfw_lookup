SFW_Lookup

Many data entry forms must restrict a field's entry to a preset list of values. In HTML forms, this is often done with drop-down lists constructed from select and option elements.

In the Schema Framework, there are several patterns that prepare drop-down selection fields. Some use the standard select/option construction, filled from various sources. Other patterns with very long lists, use a custom element that filters the list according to a partially-typed response.

Each of the pages in the following list demonstrate one method of creating a limited-option entry element. Some additional information will be offered on each page, but for the most part, the intended focus of comparison and learning is in the code of the SQL and SRM files of each example.
